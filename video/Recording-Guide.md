# Recording Guide (MP4, <= 4 minutes)

Suggested flow:
1. Intro slide: project name, team members (10s)
2. CLI UI demo (2 min)
   - `new`, `name random`, `show`
   - basic `move` and illegal move example
   - `undo` demonstration
   - `record start`, a few moves, `record stop`
   - `replay <file.record>`
   - `save` and `load`
3. Design highlights (1 min)
   - MVC and rules enforcement
4. Lesson learned (30s)
5. Closing (10s)

Tools:
- Use OBS or PowerPoint Recorder to capture terminal window
- Export MP4 at 1080p, 30fps
- Keep total duration <= 4 minutes

Deliver:
- Put the MP4 into `video/` folder, e.g., `video/demo.mp4`
