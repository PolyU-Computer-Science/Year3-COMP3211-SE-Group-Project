# Jungle (Dou Shou Qi) CLI Game

This repository contains a complete command-line implementation of the Jungle board game for COMP3211 (Fall 2025), including:
- Model package (`model/`) separated for unit testing
- CLI interface (`cli/`) with rich commands
- Unit tests (`tests/`) and coverage script using Python stdlib
- Documentation (`docs/`): SRS, Design, Developer/User manuals, Slides outline
- Sample saves/records (`records/`)
- Recording guide (`video/`)

Quick start (Windows / Python 3.11+):
- Run game: `python -m cli.main`
- Run tests: `python -m unittest`
- Coverage report: `python run_tests_with_coverage.py`

See `docs/Developer-Manual.md` and `docs/User-Manual.md` for details.
